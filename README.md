# 期末粥 (Final-Exam-Porridge)

> "喂我复习资料，还你一碗热乎的备考精华。"

**期末粥** 是一个通用的 AI Agent Skill，适用于任何支持 Markdown 格式 Skill/Agent 指令的 AI 编程助手（如 WorkBuddy、Trae、Cursor、Claude Code、GitHub Copilot 等）。把课件、笔记、教材甚至上课录音丢进来，自动提取考试重点、构建知识体系图谱、按实际考试题型分类训练、生成贴合真实考试的模拟卷，最终输出排版精美的 PDF 备考文件 + 量身定制的学习方案。

---

## 功能流程

| 步骤 | 做什么 |
|------|--------|
| 0 | 确认科目、考试类型、剩余天数、**老师是否给了题型分布** |
| 1 | 解析所有格式资料：PPT、Word、PDF、图片、视频、音频 |
| 2 | 提取知识点 + 构建**知识体系图谱**（脑图、依赖链、学习路线） |
| 3 | 在线搜索补充资料，深入讲解每个知识点（定义、记忆法、易错点、出题角度） |
| 4 | 搜索 B 站速成视频，为每个核心章节附上视频链接 |
| 5 | 生成结构化知识库（Markdown 文件） |
| 6 | **按实际考试题型**分类训练：选择、简答背诵版、综合等（只出考试考的题型） |
| 7 | 生成**贴合真实考试**的模拟卷（题型/题量/分值完全一致）+ 参考答案 + 评分标准 |
| 8 | 导出精美 **PDF** 备考文件 |
| 9 | 生成**逐日学习方案** + 每日背诵计划 |

---

## 支持的输入格式

- PPT (.pptx) — 提取文字 + 备注
- Word (.docx) — 检测格式（加粗、颜色、高亮）自动标记老师重点
- PDF — 读取全文结构
- 图片 (JPG/PNG) — 多模态识别文字
- 视频 (MP4) — 提取音频后转文字
- 音频 (MP3/WAV) — 语音转文字，识别关键词

## 核心设计理念

### 题型跟老师走

这是期末粥最核心的规则：

- **老师给了题型分布** → 题型训练和模拟卷严格按老师的来，不擅自添加没考的题型
- **老师没给题型分布** → 默认全题型覆盖（选择、填空、简答、计算、综合）

例如老师给了「单选 15 题 + 简答 6 题 + 综合 4 题」，期末粥只会出这 3 种题型的训练和模拟卷，不会画蛇添足加填空或计算题。

### 知识体系化

知识点不是孤立罗列的——每个知识点标注前置知识、后续知识、跨章节关联，绘制完整知识体系图谱，让零散的知识串成一张网。

### 简答题可背诵

简答题答案写成精炼的条目化格式，标注背诵关键词和记忆口诀，拿到就能直接在考场上默写。

### 综合题 3 题制

每种综合题型至少 3 道：理解型（详细讲解）→ 巩固型（换数据）→ 自练型（只给答案）。

### 纯文本符号，不用 Emoji

全部使用 ★ [▽] [▼] 【!!注意!!】等纯文本符号，确保 PDF 排版不乱码。

## 生成的文件结构

```
[科目名称]备考知识库/
├── 00_学习方案.md                  # 逐日学习计划 + 背诵日程
├── 01_考试重点总览.md               # 所有知识点速览 + 分值预估
├── 02_知识体系图谱.md               # 脑图 + 章节依赖 + 学习路线
├── 03_知识点详解/                   # 按章节深入讲解
│   ├── 第一章_XXX.md
│   └── ...
├── 04_题型训练/                     # 只出考试实际考的题型
│   ├── 选择题训练.md                # 考试有选择 → 生成
│   ├── 简答题背诵版.md              # 考试有简答 → 生成（可直接背诵）
│   └── 综合题训练.md                # 考试有综合 → 生成（每种3题）
├── 05_模拟试卷.md                   # 题型/题量/分值严格匹配真实考试
├── 06_速成视频合集.md               # B 站视频链接汇总
└── 07_核心公式速查.md               # 考前快速回顾
```

## 安装方式

### 方式一：下载 zip 导入

1. 从 [Releases](../../releases) 下载最新 `期末粥.zip`
2. 导入到你的 AI Agent 工具中（WorkBuddy 技能导入 / Trae Skill 导入 / Cursor Rules 导入等）

### 方式二：git clone

```bash
# 克隆到你的 AI 编程助手的 Skills/Rules 目录，例如：
# WorkBuddy:   ~/.workbuddy/skills/
# Trae:        ~/.trae/skills/
# Cursor:      .cursor/rules/
# Claude Code: ~/.claude/skills/

git clone https://github.com/YOUR_USERNAME/期末粥.git 你的Skills目录/期末粥
```

### 兼容性

| 工具 | 支持方式 |
|------|---------|
| WorkBuddy | 原生 Skill 格式，直接导入 zip 或放 `~/.workbuddy/skills/` |
| Trae | 放 `~/.trae/skills/` 目录，.trae.md 规则兼容 |
| Cursor | 放 `.cursor/rules/` 目录，作为 Project Rules 使用 |
| Claude Code | 放 `~/.claude/skills/` 或通过 CLAUDE.md 引用 |
| GitHub Copilot | 放 `.github/copilot-instructions.md` 或作为自定义指令 |

## 依赖环境

- **Python 3.10+**，需安装以下包：
  ```bash
  pip install python-pptx python-docx reportlab
  ```
- 可选：`ffmpeg`（视频音频提取）、`openai-whisper`（本地语音转文字）

## Skill 目录结构

```
期末粥/
├── SKILL.md                          # Skill 定义与工作流（核心指令）
├── README.md                         # 本文件
├── references/
│   └── knowledge_base_template.md    # 知识库结构与格式参考
├── scripts/
│   ├── extract_ppt.py                # PPT 文字 + 备注提取
│   ├── extract_docx.py               # Word 解析（含格式检测）
│   ├── extract_audio.py              # 视频音频提取
│   ├── transcribe_audio.py           # 语音转文字（Whisper / OpenAI API）
│   └── generate_pdf.py               # Markdown 知识库 → PDF
└── assets/                           # （预留）
```
